Detailed Description

The Gui Elem's implementation.

A Gui is an Elem that should only be instancied once. It is responsable for holding and redenrering every others widget. This Elem implementation is usefull if you need your GUI to only occupy a part of the screen, as an overlay for exemple. It use an already existing SDL_Renderer, and a predefined size, as well as a transparent background to be render on top of everything else. If you dont need to manually render anything else, you should consider using a Root instead.
