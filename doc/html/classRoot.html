Detailed Description

The Root Elem's Implementation.

A Root is an Elem managing the SDL_Window and SDL_Renderer for you. It should only be instancied once. By just constructing it with a size, it will create a resizable window and its render for you, then manage the rendering of every widgets you attach to it. Its easy to use if you dont want to manually use the SDL. If you do however, you should consider using a Gui instead.
